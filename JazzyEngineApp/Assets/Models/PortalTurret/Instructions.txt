Blender:
	1.When you import the model, make sure to use the "Auto Smooth" option with the angle set to 30 degrees,
	  the model will use the its correct normals if you checkmark this option. If not , it will use the 
	  incorrect normals.
	2.Go to the shading workspace, click on the texture node of each material on the model, open its properties
	  by pressing N, go to the item tab, and set the Alpha to Channel Packed.
